# 📘 Exercícios de JavaScript

## 🧠 Objetivo
Praticar estruturas de repetição, validação e interação com o usuário.

## 📌 Exercícios

1. Pedir um número positivo e continuar solicitando até ser válido.  
2. Receber notas (0 a 10) até um número negativo e exibir a média.  
3. Somar valores de compras até o usuário digitar 0 e mostrar o total.  
4. Pedir nome e sobrenome; repetir se algum estiver vazio.  
5. Simular saque com saldo inicial, permitindo saque apenas com saldo suficiente até zerar.

## 💻 Tecnologias
- JavaScript  
- `while`, `if/else`, `prompt`, `alert`

## 🚀 Como executar
Abra o código em um navegador ou rode em um ambiente como VS Code.
